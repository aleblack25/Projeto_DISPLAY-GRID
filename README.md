# 🧮 Calculadora Web

Projeto de uma calculadora desenvolvido com **HTML, CSS e JavaScript**, utilizando **CSS Grid** para organizar os botões e a interface.

## 📋 Funcionalidades

* Adição de números e operadores
* Operações matemáticas básicas:

  * Soma (+)
  * Subtração (-)
  * Multiplicação (*)
  * Divisão (/)
* Botão de limpar tela (AC)
* Inversão de sinal (+/-)
* Suporte a números decimais
* Resultado do cálculo através do botão "="

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* CSS Grid

## 📂 Estrutura do projeto

```bash
📦 calculadora
 ┣ 📜 index.html
 ┣ 📜 style.css
 ┣ 📜 script.js
 ┗ 📜 README.md
```

## 🎨 Interface

A interface foi inspirada em calculadoras modernas, utilizando:

* Layout responsivo
* Organização com Grid Layout
* Botões com cores diferenciadas
* Display para visualização das operações

## 💻 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://aleblack25.github.io/Projeto_DISPLAY-GRID/
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` no navegador.
